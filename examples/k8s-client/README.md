# k8s-client

Example for usage of the vks-k8s-auth package as k8s-client.

# How To

Export SUPERVISOR_ENDPOINT, VSPHERE_USERNAME and VSPHERE_PASSWORD as env.  
Execute main.go:

```
export SUPERVISOR_ENDPOINT=10.24.68.5
export VSPHERE_USERNAME=admbaumann@soultec.lab
export VSPHERE_PASSWORD=Ame9YZt8p6WN8tn%EH7c
go run main.go
```
